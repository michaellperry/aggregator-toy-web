Getting Started

Installation

npm install aggregator-toy

Basic Usage

Create a pipeline to transform and aggregate your data:

import { createPipeline } from 'aggregator-toy';

// Define your input type
interface Sale {
    category: string;
    product: string;
    price: number;
}

// Create a pipeline that groups by category and sums prices
const pipeline = createPipeline<Sale>()
    .groupBy(['category'], 'items')
    .sum('items', 'price', 'totalRevenue')
    .build(setState, typeDescriptor);

// Add items to the pipeline
pipeline.add('sale1', { category: 'Electronics', product: 'Phone', price: 500 });
pipeline.add('sale2', { category: 'Electronics', product: 'Laptop', price: 1200 });
pipeline.add('sale3', { category: 'Clothing', product: 'Shirt', price: 50 });

// Output: [
//   { category: 'Electronics', totalRevenue: 1700, items: [...] },
//   { category: 'Clothing', totalRevenue: 50, items: [...] }
// ]

API Reference

PipelineBuilder Methods

groupBy(keyProperties, arrayName)

Groups items by the specified key properties, collecting non-key properties into a named array.

.groupBy(['category', 'region'], 'items')

sum(arrayName, propertyName, outputProperty)

Sums a numeric property over items in a nested array.

.sum('items', 'price', 'totalPrice')

count(arrayName, outputProperty)

Counts items in a nested array.

.count('items', 'itemCount')

min(arrayName, propertyName, outputProperty)

Finds the minimum value of a property in a nested array.

.min('items', 'price', 'minPrice')

max(arrayName, propertyName, outputProperty)

Finds the maximum value of a property in a nested array.

.max('items', 'price', 'maxPrice')

average(arrayName, propertyName, outputProperty)

Computes the average of a numeric property in a nested array.

.average('items', 'price', 'avgPrice')

pickByMin(arrayName, propertyName, outputProperty)

Picks the object with the minimum value of a property from a nested array.

.pickByMin('items', 'price', 'cheapestItem')

pickByMax(arrayName, propertyName, outputProperty)

Picks the object with the maximum value of a property from a nested array.

.pickByMax('items', 'price', 'mostExpensiveItem')

commutativeAggregate(arrayName, propertyName, add, subtract)

Creates a custom commutative aggregate with add and subtract operators.

.commutativeAggregate(
    'items',
    'total',
    (acc, item) => (acc ?? 0) + item.value,
    (acc, item) => acc - item.value
)

in(...pathSegments)

Creates a scoped builder that applies operations at the specified path depth.

.in('cities').sum('venues', 'capacity', 'totalCapacity')

defineProperty(propertyName, compute)

Defines a computed property on each item.

.defineProperty('fullName', item => `${item.firstName} ${item.lastName}`)

dropProperty(propertyName)

Removes a property from each item.

.dropProperty('temporaryField')

Examples

Nested Grouping with Aggregates

interface Venue {
    state: string;
    city: string;
    venue: string;
    capacity: number;
}

const pipeline = createPipeline<Venue>()
    .groupBy(['state', 'city'], 'venues')
    .groupBy(['state'], 'cities')
    .in('cities').count('venues', 'venueCount')
    .in('cities').sum('venues', 'capacity', 'totalCapacity');

pipeline.add('v1', { state: 'TX', city: 'Dallas', venue: 'Arena', capacity: 20000 });
pipeline.add('v2', { state: 'TX', city: 'Dallas', venue: 'Stadium', capacity: 50000 });
pipeline.add('v3', { state: 'TX', city: 'Houston', venue: 'Center', capacity: 18000 });

// Output:
// [{
//   state: 'TX',
//   cities: [
//     { city: 'Dallas', venueCount: 2, totalCapacity: 70000, venues: [...] },
//     { city: 'Houston', venueCount: 1, totalCapacity: 18000, venues: [...] }
//   ]
// }]

Custom Aggregate for Min/Max Range

interface MinMax {
    min: number;
    max: number;
}

const pipeline = createPipeline<{ category: string; rating: number }>()
    .groupBy(['category'], 'products')
    .min('products', 'rating', 'lowestRating')
    .max('products', 'rating', 'highestRating');

Incremental Updates

// Add an item
pipeline.add('item1', { category: 'A', value: 100 });
// totalValue = 100

// Add another item
pipeline.add('item2', { category: 'A', value: 50 });
// totalValue = 150 (incrementally updated)

// Remove an item
pipeline.remove('item1');
// totalValue = 50 (decremented)

Type System

Aggregator Toy provides strong type inference throughout the pipeline:

KeyedArray

type KeyedArray<T> = { key: string, value: T }[];

The fundamental data structure used for grouping items with their unique keys.

Type Transformation

The type system automatically tracks transformations through the pipeline:

// Input type
interface Sale { category: string; product: string; price: number; }

// After groupBy(['category'], 'items')
// Type becomes: { category: string; items: KeyedArray<{ product: string; price: number }> }

// After sum('items', 'price', 'total')
// Type becomes: { category: string; items: KeyedArray<...>; total: number }
